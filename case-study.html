The Project: H.C. Lombardo NFL Analytics Dashboard

For my database class (IS330), I've been building an NFL analytics dashboard. Named it after Vince Lombardi because a football app needs a football name. I've been working on it for about 4 weeks using GitHub Copilot, and I've been keeping detailed notes the whole time because I wanted to document what working with AI actually feels like.

What I'm studying: How my collaboration with AI changed over time. How trust developed, how I learned to communicate differently, what the experience is like compared to coding without AI. Using my own experience as data‚Äîthat's called autoethnography.
